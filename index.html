Smelt: Batch Package Manager

Windows Installation

Install this package manager

  1. Download setup.bat.
  2. Double-click it.
  3. You're done! There will be:
    • two new folders:
      extension and logs
    • and five new files:
      sbpm.bat, install.bat, uninstall.bat, update.cmd, and reinstall.bat

To install commands pre-downloaded from other sources:

  1. Move the downloaded commands into the extensions folder.
  2. Double-click install.bat
  3. You're done! A log will be available at logs\install.log.

To install commands available here:

  1. Open Command Prompt (Win+R, type cmd and hit Enter/Win+X, C)
  2. Type sbpm install <package> (replace <package> with the name of the desired package.)
  3. You're done! A log will be available at logs\install.log.

To upgrade commands pre-downloaded from other sources:

  1. Delete the previous versions from the extension folder.
  2. Move the new versions into the extension folder.
  3. Double-click reinstall.bat
  4. You're done! Logs will be available at logs\uninstall.log and logs\install.log.

To upgrade commands available here:

  1. Open Command Prompt (Win+R, type cmd and hit Enter/Win+X, C)
  2. Type sbpm install <package> /upgrade (replace <package> with the name of the desired package.)
  3. You're done! A log will be available at logs\install.log.

To uninstall SBPM completely:

  1. Double-click uninstall.bat
  2. You're done! A log will be available at logs\uninstall.log.
  3. :(

To uninstall a command installed with SBPM:

  1. Open Command Prompt (Win+R, type cmd and hit Enter/Win+X, C)
  2. Type sbpm uninstall <package> (replace <package> with the name of the desired package.)
  3. You're done! A log will be available at logs\uninstall.log.

Update the package manager

  1. Open Command Prompt (Win+R, type cmd and hit Enter/Win+X, C)
  2. Type sbpm update
  3. Wait for it to go through the process of updating.
  4. You're done! Logs will be available at logs\install.log, logs\uninstall.log, and logs\update.log

Known issues

    #1 Windows SmartScreen may prevent setup.bat from running. To get around this, right click it, choose "Properties", check "Unblock", and click OK.
